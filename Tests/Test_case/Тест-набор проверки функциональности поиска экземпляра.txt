		2.1 Tests/search_instance.py::Test_search_instance::test_negative_search_instance[data_negative_test0]

			Описание: Сервер должен принимать только корректные значения передаваемого свойства name

			Метод: GET

			params={}
			
			Ожидаемый результат : Код ответа на запрос - 400, Ошибка : 'name parameter is required'


		2.2 Tests/search_instance.py::Test_search_instance::test_negative_search_instance[data_negative_test1]

			Описание: Сервер должен принимать только корректные значения передаваемого свойства name.
				Отработка сценария НЕ нахождения экземпляра 

			Метод: GET

			url_request=url + `?name=not+find+name`
			
			Ожидаемый результат : Код ответа на запрос - 400, Ошибка : 'No such name'


		2.3 Tests/search_instance.py::Test_search_instance::test_negative_search_instance[data_negative_test2]

			Precondition : Создать экземпляр с параметрами = {'name':'not allowed without plus'} 


			Описание: Сервер должен принимать только корректные значения передаваемого свойства name.
				Если name состоит из нескольких слов - в запросе поиска между словами необходимо вставлять знак +

			Метод: GET

			url_request=url+?name=not allowed without plus
			
			Ожидаемый результат : Код ответа на запрос - 400, Ошибка : 'No such name'
